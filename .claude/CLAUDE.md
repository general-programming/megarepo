# Word Blender Rules

## Git Commits

- After completing a full task, **always** use `AskUserQuestion` to ask the user if they want to commit. Do not skip this step.
- Every commit message must include this at the end. Replace Assisted-By depending on your identity.

  ```text
  Assisted-By: Claude-Sonnet-4-6
  ```

- Stage only relevant files; never commit secrets.
